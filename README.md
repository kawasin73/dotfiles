## Commands

create ssh key

```bash
mkdir ~/.ssh && cd ~/.ssh
ssh-keygen
```

setup my mac

```bash
git clone git@github.com:kawasin73/dotfiles.git ~/dotfiles
cd ~/dotfiles
make init
```
